GitHub Copilot Enterprise — Sessão 1, Parte 1

Nome: _________________________________
Data: 11/06/2026
Turma: ________________________________
1. Instalação do VS Code

Acessei o site code.visualstudio.com

Abri o site no navegador e localizei o botão de download

Baixei o instalador do VS Code para Windows

Cliquei em "Download for Windows" e o arquivo .exe foi baixado

Executei o instalador até o final

Avancei por todas as telas do instalador e marquei "Adicionar ao PATH"

O VS Code abriu corretamente

Encontrei o VS Code no menu Iniciar e ele abriu sem erros

2. Instalação das Extensões

Abri o painel de extensões (Ctrl+Shift+X)

Cliquei no ícone 🧩 na barra lateral esquerda

Instalei a extensão "GitHub Copilot"

Pesquisei, encontrei e cliquei em Instalar — extensão principal da IA

Instalei a extensão "GitHub Copilot Chat"

Pesquisei, encontrei e cliquei em Instalar — extensão do painel de chat

3. Login com Conta Corporativa

Cliquei em "Sign in to GitHub" quando solicitado

O VS Code pediu login após instalar as extensões

Entrei com a conta do GitHub da organização da empresa

É a conta github.com que recebeu o convite da organização — não o e-mail do Windows nem o e-mail corporativo

4. Verificação e Teste

O ícone do Copilot está ativo no canto inferior direito

Ícone colorido (não cinza nem com X)

O painel de chat 💬 aparece na barra lateral

Consigo abrir o chat do Copilot

Fiz meu primeiro pedido ao Copilot

Pedi "Crie um programa Hello World em português" e o Copilot respondeu

⚠️ Problemas? Se o ícone do Copilot aparecer cinza ou com X, clique sobre ele e selecione "Sign in to GitHub". Use sempre o e-mail corporativo @everymind.com.br.
