📄 O que é um arquivo CSV?

📊

CSV = Planilha do Excel em formato texto

Imagine uma tabela do Excel, mas salva como texto simples, onde cada vírgula separa uma coluna. É o formato mais comum para trocar dados entre sistemas.

Como parece no arquivo:

📄 dados.csv (texto puro)
Nome,Idade,Cidade
João,30,São Paulo
Maria,25,Rio de Janeiro
Pedro,40,Curitiba

Como o Copilot vai transformar:

NomeIdadeCidade
João30São Paulo
Maria25Rio de Janeiro
Pedro40Curitiba
CSV é amplamente usado em exportações do Excel, sistemas de RH, relatórios de vendas — você provavelmente já tem arquivos CSV no trabalho!

🎯 Passo 1 — Transformar em Tabela HTML

Com o arquivo dados.csv aberto no VS Code, vá ao Copilot Chat e use este prompt no Agent Mode:

🤖 Agent Mode — Copie e cole este prompt:
Leia o arquivo dados.csv e gere uma tabela HTML com todos os dados. A tabela deve ter cabeçalho em negrito, bordas visíveis e uma linha de rodapé mostrando o total de registros.

O que o Copilot vai fazer:

1. Lê o CSV

Entende a estrutura dos dados automaticamente

2. Cria o HTML

Gera uma tabela visual completa com estilos

3. Salva o arquivo

Cria um novo arquivo resultado.html no projeto

Abra o arquivo resultado.html gerado no navegador para ver a tabela — clique com botão direito → "Open with Live Preview"

🎯 Passo 2 — Converter para JSON

Ainda com o dados.csv aberto, use este prompt no Agent Mode:

🤖 Agent Mode — Copie e cole este prompt:
Converta o arquivo dados.csv para o formato JSON. Cada linha deve virar um objeto com os campos Nome, Idade, Cidade, Departamento e Salario. Salve o resultado como dados.json.

O CSV assim:

Nome,Idade,Cidade
João,30,São Paulo
Maria,25,Rio

Vira JSON assim:

[
{"Nome": "João",
"Idade": 30,
"Cidade": "São Paulo"},
{"Nome": "Maria", ...}
]
JSON é o formato mais usado para trocar dados entre sistemas digitais. Útil para integrar com APIs, painéis de BI e outros sistemas da empresa.

🎯 Passo 3 — Estatísticas dos Dados

Nosso terceiro e último prompt para este exercício:

🤖 Agent Mode — Copie e cole este prompt:
Analise o arquivo dados.csv e crie um relatório com estatísticas básicas: total de pessoas por cidade, total por departamento, média de idade e faixa de salário (menor e maior). Salve como relatorio-estatisticas.txt.

Resultado esperado no arquivo:

📊 Relatório de Estatísticas — dados.csv

Média de idade: 31,6 anos
Salário menor: R$ 4.200 | Maior: R$ 7.500
Por Departamento: Marketing (2), Vendas (1), TI (1), RH (1)
Por Cidade: São Paulo, Rio, Curitiba, BH, Porto Alegre
