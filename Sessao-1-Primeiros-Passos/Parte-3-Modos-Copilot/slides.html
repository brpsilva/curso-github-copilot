Os 3 Modos do Copilot

A parte mais importante do treinamento — entenda como "falar" com a IA

Ao final desta parte você vai saber usar:
  • Ask Mode — para perguntar e aprender
  • Agent Mode — para criar coisas do zero e executar tarefas
  • Plan Mode — para planejar antes de executar

⚡ Os 3 Modos em Um Olhar

💬
Ask Mode
Pergunte — aprenda — tire dúvidas
"O que é um arquivo CSV?"
🤖
Agent Mode
Execute tarefas completas do zero
"Crie um programa que leia CSV"
📋
Plan Mode
Planeje antes de executar
"Crie um plano para meu projeto"
Você vai usar os 3 modos hoje nos exercícios práticos. Não precisa memorizar agora — vai ficando natural com o uso!
💬

Ask Mode

Use para perguntar, aprender e tirar dúvidas

É como ter um especialista ao seu lado. Você pergunta qualquer coisa e o Copilot explica em linguagem simples.

Exemplos de prompts:

💬 Ask Mode:
Explique o que é este arquivo.
💬 Ask Mode:
O que é um arquivo CSV?
💬 Ask Mode:
Como eu poderia melhorar este email?

Quando usar:

📚 Para aprender

Entender conceitos novos sem sair do VS Code

🔍 Para revisar

Pedir para o Copilot analisar e avaliar seu trabalho

❓ Para tirar dúvidas

Qualquer pergunta que você teria para um colega especialista

🤖

Agent Mode

Use para criar e executar tarefas completas do zero

O mais poderoso dos modos. O Copilot age como um assistente autônomo — cria arquivos, faz múltiplas etapas e executa tudo sozinho.

Exemplos de prompts:

🤖 Agent Mode:
Leia o arquivo dados.csv e gere uma tabela HTML com os dados.
🤖 Agent Mode:
Crie um email marketing para o produto descrito no arquivo.
🤖 Agent Mode:
Analise este CSV e crie um relatório com estatísticas básicas.

O que o Agent faz:

📄 Cria arquivos novos

Gera o resultado diretamente como arquivo no seu projeto

🔀 Executa múltiplas etapas

Faz tudo sozinho: lê, processa, escreve o resultado

🔄 Trabalha em sequência

Pode encadear várias tarefas em um único pedido

📋

Plan Mode

Use para planejar antes de começar a executar

Antes de pedir para o Copilot fazer algo grande, use o Plan Mode para discutir a estratégia e validar o caminho.

Exemplos de prompts:

📋 Plan Mode:
Crie um plano para transformar nossos dados de vendas em um relatório mensal.
📋 Plan Mode:
Quais passos eu precisaria seguir para criar uma campanha de email marketing?
📋 Plan Mode:
Faça um plano para organizar os arquivos do nosso departamento.

Quando usar:

🏗️ Projetos maiores

Antes de pedir uma tarefa complexa, planeje as etapas

🤝 Discussão em equipe

Apresentar o plano para o time antes de executar

✅ Validação antecipada

Confirmar que o Copilot entendeu o que você quer antes de começar

⚖️ Comparação: Quando Usar Cada Modo

Modo Use quando você quer... Exemplo prático
💬 Ask Aprender, tirar dúvidas, pedir análise "O que este texto significa?" / "Como está meu email?"
🤖 Agent Criar algo novo do zero, tarefa completa "Crie um relatório a partir deste CSV"
📋 Plan Planejar etapas antes de executar "Qual o melhor jeito de montar esta campanha?"
Regra prática: não sabe qual usar? Comece sempre pelo Ask Mode — ele ajuda a esclarecer o que você precisa fazer!
