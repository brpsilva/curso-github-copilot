🧠 Guia Rápido — Os 4 Modos do GitHub Copilot

Sessão 1, Parte 3 — Cole na sua mesa ou salve como referência

💬
Ask Mode
Perguntar • Aprender • Analisar
Prompts típicos:
  • O que é um arquivo CSV?
  • Explique este texto para mim.
  • Como eu poderia melhorar este email?
  • O que significa esta mensagem de erro?
  • Quando quiser aprender algo novo
  • Quando quiser revisar um trabalho
  • Quando quiser uma opinião da IA
✏️
Edit Mode
Modificar • Corrigir • Transformar
Prompts típicos:
  • Transforme este texto em HTML.
  • Corrija erros de português.
  • Torne este email mais formal.
  • Adicione uma introdução a este texto.
  • Quando o arquivo já existe
  • Quando quer mudar o formato
  • Quando quer refinar um rascunho
🤖
Agent Mode
Criar • Executar • Automatizar
Prompts típicos:
  • Leia o CSV e gere uma tabela HTML.
  • Crie um email marketing com base neste texto.
  • Analise os dados e gere estatísticas.
  • Crie um relatório completo do projeto.
  • Quando quer criar algo do zero
  • Quando a tarefa tem várias etapas
  • Quando quer automação completa
📋
Plan Mode
Planejar • Estruturar • Validar
Prompts típicos:
  • Crie um plano para organizar meus arquivos.
  • Que etapas devo seguir para criar X?
  • Qual a melhor abordagem para este projeto?
  • Ajude-me a estruturar uma campanha.
  • Antes de começar projetos grandes
  • Quando quer discutir estratégia
  • Quando quer validar o caminho
