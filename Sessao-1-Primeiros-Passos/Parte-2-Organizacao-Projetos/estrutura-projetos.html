📌 Para que serve cada pasta?

📂 Projeto-CSV

Para trabalhar com planilhas e dados tabulares. Coloque aqui seus arquivos .csv e os resultados gerados pelo Copilot.

📂 Projeto-Texto

Para criação e transformação de textos. Emails, posts, resumos, relatórios em texto.

📂 Projeto-Marketing

Para materiais de comunicação, campanhas e conteúdo para redes sociais gerados com a IA.

📂 Projeto-Testes

Espaço livre para experimentar novas ideias, testar prompts e explorar as capacidades do Copilot.

📏 Regras de Ouro para Nomes de Arquivo

  • Use hífens (-) para separar palavras: email-marketing-junho.html
  • Use letras minúsculas sempre que possível: relatorio-vendas.txt
  • Inclua datas quando o arquivo vai mudar com o tempo: reuniao-2026-06-11.txt
  • Seja descritivo: olhando o nome, qualquer pessoa entende o que tem dentro
  • Evite espaços em nomes de arquivos: meu arquivo.txt → use meu-arquivo.txt
  • Evite acentos e cedilha em nomes: relatório.txt → use relatorio.txt
  • Nunca use versões no nome: final_v2_certo_DEFINITIVO.txt
