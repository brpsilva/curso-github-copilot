Exercício Prático: Criação de Texto

Um texto simples — três formatos diferentes em minutos

O que você vai criar neste exercício:
  • Um email marketing profissional
  • Um post otimizado para LinkedIn
  • Um resumo executivo em 3 tópicos

📄 Nosso Texto de Partida

Abra o arquivo texto-base.txt da pasta Projeto-Texto no VS Code. Este é o conteúdo que vamos transformar:

📄 texto-base.txt — texto original
A empresa lançou um novo produto focado em inteligência artificial chamado Everymind AI Assistant. O produto utiliza tecnologia de processamento de linguagem natural para automatizar tarefas repetitivas no ambiente corporativo. Entre as funcionalidades principais estão: análise automática de documentos, geração de relatórios, resposta a emails e organização de dados. O produto está disponível para empresas de médio e grande porte a partir de junho de 2026, com planos mensais e anuais.

📧 Prompt 1

Transformar em email marketing profissional

💼 Prompt 2

Criar post otimizado para LinkedIn

📋 Prompt 3

Resumir em 3 tópicos executivos

📧 Prompt 1 — Email Marketing Profissional

Com o arquivo texto-base.txt aberto, use o Edit Mode e aplique este prompt:

✏️ Edit Mode — Copie e cole este prompt:
Transforme este texto em um email marketing profissional para ser enviado a clientes corporativos. O email deve ter: assunto atrativo, saudação personalizada, apresentação do produto em linguagem de negócios, 3 benefícios em destaque, chamada para ação (call to action) e assinatura da Everymind Ai/r.
✉️
📧 Resultado esperado (exemplo):

Assunto: Automatize suas tarefas com Inteligência Artificial — Conheça o Everymind AI Assistant

Prezado(a) [Nome],

Temos o prazer de apresentar o Everymind AI Assistant, nossa solução de IA desenvolvida para transformar a produtividade da sua empresa...

✅ Analisa documentos automaticamente
✅ Gera relatórios sem esforço manual
✅ Responde emails e organiza dados

Agende uma demonstração gratuita →

💼 Prompt 2 — Post para LinkedIn

Ainda com o arquivo aberto, agora use o Edit Mode com este novo prompt:

✏️ Edit Mode — Copie e cole este prompt:
Transforme este texto em uma postagem para o LinkedIn. O post deve ser envolvente, ter no máximo 200 palavras, usar emojis estrategicamente, terminar com uma pergunta para engajar a audiência e incluir 5 hashtags relevantes sobre IA e tecnologia.
👔
💼 Resultado esperado (exemplo):

🚀 Inteligência Artificial chegando ao seu dia a dia corporativo!

Acabamos de lançar o Everymind AI Assistant — uma ferramenta que usa IA para automatizar tarefas que consomem horas do seu time...

🤖 Análise automática de documentos
📊 Relatórios gerados em segundos
✉️ Respostas de email inteligentes

Qual tarefa repetitiva você mais gostaria de automatizar no seu trabalho?

#InteligenciaArtificial #IA #Produtividade #Inovacao #EverymindAir

📋 Prompt 3 — Resumo em 3 Tópicos

Terceiro e último prompt deste exercício — use o Ask Mode:

💬 Ask Mode — Copie e cole este prompt:
Resuma este texto em exatamente 3 tópicos objetivos, no estilo de uma apresentação executiva. Cada tópico deve ter um título em negrito e duas frases de explicação. Use linguagem formal e direta.
📝
📋 Resultado esperado (exemplo):

1. O Produto: O Everymind AI Assistant é uma solução de inteligência artificial para automatização de tarefas corporativas. Utiliza tecnologia avançada de processamento de linguagem natural.

2. Funcionalidades: O produto oferece análise de documentos, geração de relatórios, resposta a emails e organização de dados. Todas as tarefas executadas de forma automatizada e inteligente.

3. Disponibilidade: Disponível para médias e grandes empresas a partir de junho de 2026. Oferece planos mensais e anuais com suporte dedicado.

🚀 Aplicação real na Everymind:

O mesmo texto pode ser reutilizado para comunicados internos, newsletters, apresentações e redes sociais — tudo com um prompt diferente. Isso é a essência do Copilot: multiplicar o que você já tem!
